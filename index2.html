Self-serving system for the Hybrid-Cloud development environment

“You can make easy the development environment build up with Phoenix server pattern by the Metrom”

Get Started HOW TO USE

Simple & easy

Metrom은 사내 또는 Public cloud 환경에서 적은 비용으로 개발환경을 쉽게 구축 할 수 있도록 도와 줍니다.

Public Cloud 환경을 하나하나 수작업으로 만들 필요가 없습니다. 또한 개발이 끝난 장비를 더 이상 방치 할 필요도 없으며, 일정 시간이 지나면 인스턴스가 삭제되어 불필요한 과금을 부담 할 필요가 없어집니다. 더불어, 개발업무를 하지 않는 시간에는 해당 인스턴스가 가동중이지 않는 상태로 변경되어 최소한의 비용으로 개발환경을 효율적으로 사용 할 수 있게 됩니다.

PaaS from Marketplace

손쉽게 제작이 가능한 Platform As A Service를 Market Place를 통해 경험 해 보세요

Application stack 형태를 Docker-compose를 이용하여 사용자가 쉽게 PaaS를 구축할 수 있습니다. ELK, NiginX+Tomcat+MySQL 등 이미 만들어져 있는 Stack set를 기반으로 새로운 구성의 Platform의 구성이 가능합니다.

Docker based

Docker container 및 docker-compose를 기반으로 구성되어 있습니다.

전체 환경 구축을 Docker를 사용함으로써 Cloud service lock-in을 방지하며, 표준화된 이미지와 구성 Set를 기반으로 쉽게 사용이 가능합니다. 또한 배포 단위를 Docker image로 구성하여 테스트 및 서비스 적용에 있어서 쉽고 빠르게 적용이 가능합니다. Docker를 활용한 on-premises 환경은 하나의 Private cloud 환경 처럼 사용할 수 있게 됩니다.

Dev. for CI/CD

CI/CD를 고려한 설계 및 개발 방법론 제시

개발 작업을 위한 환경이 배포를 위한 단위로 재구성되고, Test 단계를 거쳐 Production 환경으로 그대로 반영이 될 수 있도록 최초 개발 단계에서부터 Continuous integration & Deploy를 고려하여 개발이 진행되도록 가이드를 제공합니다.

Phoenix Server pattern

Immutable infrastructure의 배포와 개발주기

개발환경 용도로 구성된 set가 하나의 immutable infrastructure의 형태로 배포가 되며, 다시 다음 개발업무 진행을 위해 같은 구성의 infrastructure가 개발환경으로 다시 제공되며 개발 생애주기를 반복 하도록 기반을 만들어 줍니다.

Multi-cloud service provider support

Cross-cloud Provider management includes on-premises

On-premises 환경을 포함하여, 다양한 Cloud computing 환경을 지원하며, 한 화면을 통해 전체 내역을 확인 및 관리할 수 있습니다.
